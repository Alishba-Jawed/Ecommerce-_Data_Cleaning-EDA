# 🛒 E-commerce Data Cleaning & Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project demonstrates the process of cleaning and analyzing a messy e-commerce dataset using Python. The dataset contained missing values, duplicate records, inconsistent entries, and incorrect data formats. After cleaning the data, Exploratory Data Analysis (EDA) was performed to uncover patterns and visualize key business insights.

---

## 🎯 Objectives

* Explore the dataset
* Handle missing values
* Remove duplicate records
* Clean inconsistent data
* Standardize data formats
* Perform Exploratory Data Analysis (EDA)
* Visualize key business insights
